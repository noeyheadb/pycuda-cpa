# pycuda-CPA
CUDA implementation of CPA(Correlation Power Analysis) using [`pycuda`](https://github.com/inducer/pycuda).

## Environment setup
- GPU with compute capability less than 1.3 are not supported.
- Make `pycuda` available by installing the CUDA toolkit and setting environment variables, etc.
- Test code of `pycuda` is [here](https://documen.tician.de/pycuda/).

## Requirements
- pycuda >= 2020.1
- numpy
